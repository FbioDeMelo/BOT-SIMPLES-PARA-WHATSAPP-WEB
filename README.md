# BOT-SIMPLES-PARA-WHATSAPP-WEB
Desenvolvido usando Python, esse script tem como automatizar envios de mensagens de forma automatica, evitando o processo repetitivo do usuario.
Para usar o BOT, é necessario seguir o passo abaixo

# Funcionalidades
Envio de mensagens para uma lista de contatos predefinida.
Personalização total do texto da mensagem.
Pausas automáticas entre os envios para tentar mitigar o risco de bloqueio.
Agendamento de envios para aumentar a estabilidade do processo.
